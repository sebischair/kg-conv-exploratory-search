Testszenario (5-7 Minuten)

Sie können hier einen Agenten zur sprachgesteuerten Suche nach Tagesnachrichten testen. Der Agent kann Ihnen sowohl einen Tagesüberblick geben als auch Nachrichten über ein bestimmtes Thema heraussuchen.

Testen Sie den Agenten in einer möglichst ruhigen Umgebung und achten Sie bitte darauf, laut und deutlich zu sprechen. Falls der Agent, nachdem Sie fertig gesprochen haben, immer noch "Bitte sprechen" anzeigt, wiederholen Sie bitte laut und deutlich Ihre Eingabe oder aktualisieren Sie die Seite. Damit der Agent optimal funktioniert, benutzen Sie bitte den Browser Google Chrome. Falls der Agent auch nach dem Aktualisieren der Seite nicht reagiert, versuchen Sie es bitte später erneut.

Um direkt zu beginnen, drücken Sie auf das Mikrofonsymbol und begrüßen Sie den Agenten mit einem "Hallo". Sie können jederzeit die Sprachausgabe unterbrechen und einen neuen Sprachbefehl eingeben, indem Sie erneut auf das Mikrofonsymbol drücken.
